Updating Theme
==============

Compile theme
-------------
`cd app/themes/bootstrap
gulp`

Installing gulp
---------------

#1 install node
On a mac you can use homebrew
`brew install node`

#2 install theme
`npm install`

#3 install gulp
`npm install --global gulp-cli`
Check out more details at https://github.com/gulpjs/gulp/blob/master/docs/getting-started.md

#4 run gulp (compile theme)
`gulp`
